Search the todo to find right place to put your changes

  1. Draw a Flowchart
  2. Write skeleton program to support the following features
  3. A random word will be chosen from at least 10 words
  4. maximum trials allowed is 6
  5. if guess is same as one of previous guess, it wont be counted as times
  6. if player guessed the word before maximum trials reached, player will won (game will show won screen)
  7. After win/lose, the guess form will be hidden and player can choose to replay
  8. when replay, a new word will be generated
  9. (Optional) Update flowchart as following requirements
  10. (Optional) Add more skeleton code
  11. (optional) Add start play screen before user play. It has a play button
  12. (optional) Player will choose a category word before starting, each category has at least ten words; There are 5 categories
